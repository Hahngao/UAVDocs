# APM飞控无人机快速使用指南

本文面向首次上手 APM/ArduPilot 飞控无人机的用户，目标是用最少步骤完成基础检查、连接、首飞与收尾。

## 1. 起飞前准备

- 检查桨叶安装方向是否正确、机臂无松动、电池电量充足
- 确认遥控器、接收机、数传/地面站、GPS、罗盘连接正常
- 在空旷、无强磁干扰、无人员聚集区域进行首飞
- 首飞建议使用 `Stabilize`、`AltHold` 或 `Loiter` 等基础模式

## 2. 上电与连接

1. 先开启遥控器，再给无人机上电
2. 等待飞控启动完成，观察状态灯是否正常
3. 连接地面站，确认能读取到：
   - 飞控姿态
   - GPS 星数
   - 电池电压
   - 遥控器通道输入
4. 若罗盘、陀螺仪、加速度计未校准，先完成校准再飞行

## 3. 起飞前必须确认

- 飞行模式切换正常
- 油门最低位时可以解锁，最高位输入正常
- GPS 定位稳定，返航点已刷新
- 电池电压在安全范围内
- 螺旋桨安装方向与电机转向匹配
- 失控保护、低电压保护参数已启用

## 4. 首飞流程

1. 将无人机平放在水平地面
2. 解锁前再次确认周围安全
3. 解锁电机
4. 缓慢推油门离地，先小范围悬停
5. 观察是否存在明显漂移、抖动、掉高或偏航异常
6. 无异常后，再进行小范围平移和模式切换测试

## 5. 降落与断电

1. 缓慢降落至地面
2. 收油门并锁定电机
3. 先断无人机电源，再关闭遥控器
4. 检查电机、电调、电池温度是否异常
5. 如有炸机、急刹、失控等情况，及时导出日志分析

## 6. 常见问题快速排查

- 不能解锁：先检查飞行模式、GPS 状态、传感器校准、遥控器通道
- 漂移明显：优先检查罗盘校准、振动、重心和桨叶安装
- 无法定点：检查 GPS 星数、罗盘方向、磁干扰环境
- 续航过短：检查电池老化、桨叶效率、整机重量和巡航油门
