# 介绍

Kerloud UAV系列作为云讷科技（深圳）面向无人机系统集成爱好者的高性价比开发平台，支持PX4、Dronecode、ROS、Nvidia和Ubuntu等高级软件社区。

## 产品指标

| Specification | Kerloud 300 | Kerloud 600 |
| --- | --- | --- |
| Diagonal rotor center distance | 30cm | 60cm |
| Endurance | 12min | 25min |
| Weight | 1.4kg with Jetson nano, an onboard camera and a battery | 2.65kg with Jetson nano, an onboard camera and a battery |
| Maximum payload | 1.2kg | 1.5kg |
